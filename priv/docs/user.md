# User

The currently logged in/authenticated User

## User Details

`GET` 

> GET user

**Permission needed:** `com.cloudflare.api.user.read`

Available in:

* free
* pro
* business
* enterprise


## Edit User

`PATCH` Edit part of your user details

> PATCH user

**Permission needed:** `com.cloudflare.api.user.update`

Available in:

* free
* pro
* business
* enterprise
