# Logcontrol CMB config for an account

## Update CMB config

**POST** `/accounts/{account_id}/logs/control/cmb/config`

Updates CMB config.


## Get CMB config

**GET** `/accounts/{account_id}/logs/control/cmb/config`

Gets CMB config.


## Delete CMB config

**DELETE** `/accounts/{account_id}/logs/control/cmb/config`

Deletes CMB config.
